Azure Web API Upload

  1. 1

    Azure portal > App Services

    Create > Select "Web App"

  2. 2

    Select Project Details

    1. Select Subscription

    2. Select/Create Resource Group

    3. Fill Instance Details like

    Web App Name, Publish as Code, Runtime stack as Dot net version

    OS as Windows, Region Central India,

    Princing plan as free f1

    Review + Create

  3. 3

    Optional

    Goto Deployment tab > Github settings > Can configure repo, branch

    It will add workflow file (YML) into our branch and whenever there is code change it will push content to our app on azure

  4. ✓

    Service created

    Goto Overview => Check "Default Domain".

  5. 5

    Goto Visual Studio

    Right Click On Project "Publish"

    Login to Azure if not already

    "+ New Profile" > Azure

    Select "Azure App Service (Windows)"

    Select "Service" from "Resource Group"

    Skip other steps > Finish > Publish at top
